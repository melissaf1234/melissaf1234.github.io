# Melissa Florence - Academic Website

This repository contains a simple static academic/personal website for Melissa
Florence Marques Miécimo da Silva.

The site is built with plain HTML and CSS. It does not require React, Node,
Jekyll, or any build tools.

## Repository Contents

- `index.html` - main website page.
- `style.css` - visual styling and responsive layout.
- `assets/melissa-florence-lattes.pdf` - Lattes CV PDF used as a downloadable
  reference.

## Editing

Edit `index.html` for content changes and `style.css` for visual changes. Since
the site is static, changes can be previewed by opening `index.html` directly in
a browser.

## GitHub Pages

The site can be served directly from the repository root.

To enable GitHub Pages:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save the configuration.

For a project repository named `trust`, GitHub Pages normally publishes at:

```text
https://melissaf1234.github.io/trust/
```

For a user-site URL such as `https://melissaf1234.github.io/`, the repository
would normally need to be named `melissaf1234.github.io`.
